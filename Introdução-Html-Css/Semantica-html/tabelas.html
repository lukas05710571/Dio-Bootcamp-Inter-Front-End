Tabelas

Uma tabela serve para organizar informações de forma estrutural ,
penssando no conceito da planilha excel nós temos linhas e colunas onde são responsáveis
pela estruta dos dados e a organização dos mesmos

Case:

Listar tabelas de pontos dos times de futebol do campeonato brasileiro

Colocação Time Pontos
1º Santos 41
2º Palmeiras 30
3º São caetano 25

O que é possível com tabelas ?

é possível adicionar bordas , imagens , e todo tipo de dados no qual você desejar.
